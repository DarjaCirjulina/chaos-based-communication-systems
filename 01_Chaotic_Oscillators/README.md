# Chaotic Oscillators

This folder contains LTspice models of chaotic oscillators and MATLAB scripts for nonlinear signal analysis.  

LTspice simulations generate time-domain waveforms exported as `.txt` files, which are processed in MATLAB.  

Implemented analysis includes cross-correlation computation, Z1Test evaluation, power spectral density, and two-dimensional attractor projections.  

These models provide the nonlinear dynamical foundation for the investigations of synchronization and communication systems presented in this repository.
