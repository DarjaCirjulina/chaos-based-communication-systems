# FM-CSK Communication System

This folder implements the simulation framework for the Frequency Modulated Chaos Shift Keying (FM-CSK) communication system.

MATLAB generates bit sequences and controls LTspice simulations of the drive and response chaotic oscillators using `.net` netlists. Waveforms are exported as `.txt` files for signal processing and performance evaluation.

The system includes additive noise modeling, a multipath channel model, demodulation procedures, and Bit Error Rate (BER) computation.

This module evaluates FM-CSK performance under realistic channel conditions, including noise and multipath propagation effects.
