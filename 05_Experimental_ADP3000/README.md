# Experimental Validation – ADP3450

This folder provides example scripts supporting experimental validation of the proposed chaos-based communication systems. The hardware implementation itself is external; this directory contains only the control and processing code.

MATLAB scripts generate bit sequences prior to transmission. Python scripts control the Analog Discovery Pro 3000 (ADP3000) for signal generation and data acquisition.

After acquisition, detection and performance evaluation are performed using post-processing scripts.

This section serves as a bridge between simulation results and hardware-based performance verification.
